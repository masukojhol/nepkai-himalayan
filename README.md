# NepkAI Himalayan Theme

A beautiful React chat interface with a Himalayan-inspired design theme. Built for Nepal's AI assistant.

## Features

- Himalayan mountain background with animated stars
- Multiple AI model support (Gemini, Groq, Claude, Codestral)
- Nepali language support with Devanagari script
- Multiple personas (General, Nibedan Writer, Coder, Loksewa Prep, CV Builder, Korean Teacher)
- Dark theme with golden accents (Kala Patthar sunrise colors)
- Responsive sidebar with chat history
- Settings and upgrade modals

## Preview

The interface features:
- Sky gradient transitioning from deep blue to golden sunrise
- Animated twinkling stars
- Mountain silhouettes with snow-capped peaks
- Glass-morphism UI elements
